# ChatGPT Sidebar Extension 🧠💬

This is a simple Chrome extension that opens [ChatGPT](https://chat.openai.com) in your browser’s sidebar, so you can chat with AI anytime — without switching tabs! 🧊⚡️📚

## Features
- Opens ChatGPT in Chrome’s new sidebar
- Easy to access while browsing
- Lightweight and minimal setup

## How to Install
1. Download or clone this repo
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer Mode** (top-right corner)
4. Click **Load unpacked** and select this folder
5. Click the puzzle icon → Pin the extension → Done! 🎉

> NOTE: You’ll need to be logged into ChatGPT in your browser to use it.
